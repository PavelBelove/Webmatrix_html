🤖 WebMatrix

Smart Data Analysis Assistant

Hello! WebMatrix is a compact tool that helps automate routine work with tables, search, and data analysis using AI. Simply open the file in your browser, upload a table, and describe in your own words what you need to find or analyze.

✨ No installation or setup required
🔒 Your data stays local
⚡ Quick results
📊 One-click analysis
🔄 Works in background
💻 Doesn't interfere with main work

🎯 What can WebMatrix do?

Company Analysis

Automatically researches websites, finds key information about business, operational scale, and specialization

Data Processing

Classifies, categorizes, and enriches tabular data with additional information

Automation

Uses AI to perform routine table tasks that you previously did manually

Validation

Checks data against specified criteria, finds inconsistencies and evidence

🔄 How does it work?

1

Upload your data table (for example, a list of companies for analysis)

2

Describe in plain language what you need to find or verify

3

WebMatrix automatically creates an optimal AI query and columns for responses

4

AI analyzes the data and adds new columns with results to your table

📚 Ready-to-use Recipes

🔍 B2B Company Analysis

Automatically determines:

  • Business size and geography
  • Main business directions
  • Partnership potential
  • International trade indicators

Example query:

Analyze the company website {{company_name}} {{website}} and evaluate:
- Scale of international operations
- Presence of currency operations
- Potential as a B2B client
- Main regions of presence

📋 Key Phrase Enhancement

Searches for key phrases in "good" signals

  • Reads quality signals
  • Identifies key phrases
  • Compiles a list

Example query:

Read the article and find key phrases that can be used to find similar news in Google:
                     - Startup raised investment
                     - Works in {{industry}}
                     - Has a distributed team
                     {{article_text}}

📊 Data Enrichment

Enhances tables with information:

  • Potential assessment
  • Feature identification

Example query:

Enhance company data with:
- Brief business description
- Main directions
- Operating specifics
- Potential opportunities

💡 Useful Tips

🎯 Analysis Accuracy

  • Use specific evaluation criteria
  • Specify priority parameters
  • Request source links for conclusions
  • Don't ask too many questions in one pass
  • Explicitly tell the model what and how it should do
  • Analyze first rows manually and compare model's answers with expected ones

⚡ Performance

  • Save successful prompts
  • Magic button can be used multiple times
  • Monitor initial results and adjust the prompt
  • Can run multiple instances in incognito mode with different API keys

📈 Improving Results

  • Check first results
  • Adjust prompt if needed
  • Include common model errors in the prompt
  • Always remember: Models can be very convincing when they lie

🚀 Quick Start

1️⃣ Get API Key

Takes just a minute:

  1. Open makersuite.google.com
  2. Click "Create API Key"
  3. Save the key in a secure place

2️⃣ Launch WebMatrix

  1. Open WebMatrix.html in browser
  2. Enter API key (saves in browser)
  3. Upload your table
  4. Describe your task in detail
  5. Click "Magic" ✨

✨ About the "Magic" Button

This is your assistant in creating effective prompts. It analyzes:

  • Your table structure
  • Data examples
  • Task description

And automatically creates an optimal prompt for analysis.

ℹ️ Important to Know

🔒 Security

API key is stored only in your browser. It's not transmitted or saved anywhere else.

💾 Data Safety

Presets and settings are stored locally. Important presets should be exported.

🌐 Accessibility

If experiencing access issues, use browser VPN extensions (Browsec, ZenMate, Urban VPN).
